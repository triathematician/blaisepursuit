Agents

Agents are the basic unit in a pursuit simulation. They have individualized behaviors and can be customized in several ways. Their basic parameters are:

  • Initial Position: Location at which the agent starts.
  • Top Speed: The maximum speed achievable by the agent.
  • Max Acceleration: The maximum rate at which the agent's speed can increase.
  • Turning Radius: The minimum radius at which the agent can turn. If 0, the agent can turn in any direction at any time. Otherwise, the agent will be unable to travel in a tighter circle than one of this radius.
  • Communications Radius: This is the maximum range at which the agent can send communications.

Agents also support more complex attributes and behaviors.

An agent's path is determined by a single main task, which includes information about the position (and velocity) of some sort of target. This may represent an actual target, or it might represent a "phantom" target, e.g. if an evader is trying to escape from two pursuers. The agent will move as close to the target as possible, subject to the restrictions imposed by its max acceleration and turning radius.
