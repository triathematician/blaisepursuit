A randomized greedy VRP construction algorithm based on the algorithm described by Clarke and Wright.

public ClarkeWright()
             throws VRPException
Constructs a randomized version of the ClarkeWright algorithm using the Randomizer object. There are no initial iterations added to the randomizer, so they must be added before use. The random number generator defaults to 'MultiplicativeCongruential'.
